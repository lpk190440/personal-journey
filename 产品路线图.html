体验升级。人脉结构化表单、社交节奏提醒、时间轴筛选栏、弹性方案预览、每日复盘入口、番茄钟专注模式、个性化设置12+项。8项功能增强。

演进策略：采用"基础补全 → AI增强 → 系统进化"三阶段路径。前两阶段在单文件架构内快速迭代，V2.0进行架构级重构。每个版本的目标覆盖率增幅控制在13-22个百分点，确保可验证、可回滚。

V1.4 - AI 全面接入
进行中

V1.5 - 深度完善
规划中
功能项描述优先级预期影响
Notification API 通知 接入浏览器通知API：任务时间前15分钟提醒、每日21:00未打卡提醒、番茄钟结束提醒、社交节奏提醒。通过Service Worker后台同步实现定时检查。 P0 高
云数据同步 JSON导入/导出升级为云端同步方案。方案A：GitHub Gist作为免费后端（用户授权后自动备份）。方案B：WebDAV（坚果云等）。自动同步间隔30分钟。 P0 高
习惯追踪模块 独立于打卡的习惯追踪系统。支持"每天阅读30分钟"类持续性行为追踪，30天完成曲线，连续天数统计，灵活周期（每天/每周N次）。 P1 中
Web Speech 语音输入 接入Web Speech API，NLP页面支持语音输入任务描述。自动转文字后走正常NLP解析链。移动端体验大幅提升。 P1 中
任务批量操作 长按多选模式：批量删除、批量改标签、批量改优先级。适合周初批量清理上周遗留任务。 P2 低
6色主题预设 在莫兰迪色系基础上提供6套配色方案（当前蓝绿/暖粉/森林/海洋/日落/暗夜），一键切换。CSS变量映射已就绪。 P2 低

V2.0 - 系统进化
远期规划
功能项描述优先级预期影响
决策引擎 智能调度系统：根据任务优先级+精力曲线+依赖关系+历史效率数据，自动生成最优时间排列方案。用户可接受/微调/拒绝。 P0 高
学习引擎 用户行为建模：学习用户的高效时段、任务偏好、精力模式。随着使用时间增长，建议越来越精准。本地轻量级模型，不依赖云端。 P1 高
监控引擎 主动干预系统：检测连续低能期（3天C档）→ 主动建议降低难度；检测连续加班 → 增加留白比例；检测社交脱节 → 提升社交任务优先级。 P1 高
画像引擎 用户能力模型：基于任务完成率、番茄钟数据、目标达成率生成"效率画像"——深度工作偏好、最佳时段、任务类型倾向、压力阈值。 P2 中
架构重构（Vite） 3906行单文件 → Vite组件化：每页面一个.vue组件，共享状态管理（Pinia），按需加载。首屏加载降至2s以内。 P0 高
IndexedDB 存储层 localStorage(5MB限制) → IndexedDB(无限制)。支持海量历史数据、快速索引查询、后台事务写入。为学习引擎提供数据基础。 P0 高
API Key 安全存储 从硬编码迁移至浏览器加密存储（Crypto API）。用户输入一次后加密保存，传输时解密。支持多AI服务商切换。 P1 中
目标智能拆解 输入模糊目标（如"学好英语"），AI+规则引擎辅助拆解为SMART目标链：年目标→季OKR→月里程碑→周任务→日行动。 P1 中

三、技术债务清单

当前技术债务
8项待清理
#债务项严重度计划版本描述与清理方案
1 单文件架构 高 V2.0 3906行单HTML包含全部逻辑。维护成本随功能增长指数级上升。V2.0迁移至Vite+Vue组件化架构。
2 API Key 硬编码 高 V1.5 API Key明文写在JS中，代码仓库泄露风险。V1.5迁移至Crypto API加密存储+用户输入。
3 localStorage 容量 中 V2.0 localStorage 5MB限制，历史数据超过6个月可能溢出。V2.0迁移至IndexedDB。
4 无虚拟滚动 中 V1.5 时间轴长列表（100+任务）直接DOM渲染，低端设备卡顿。V1.5引入虚拟滚动。
5 无测试覆盖 中 V2.0 零单元测试、零E2E测试。V2.0组件化后引入Vitest+Playwright，核心方法80%覆盖。
6 CSS 内联无设计系统 低 V2.0 CSS变量已定义但无原子类/组件样式规范。V2.0建立Design Token系统+Tailwind原子类。
7 Alpine.js 版本锁定 低 V2.0 V2.0迁移至Vue 3 Composition API时自然解决。当前Alpine.js 3.14.8功能满足需求。
8 Service Worker 粗粒度 低 V1.5 当前SW缓存策略过于简单（HTML=Network First）。V1.5优化为版本化缓存+增量更新。
