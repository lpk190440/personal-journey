## 一、系统定位与核心功能描述

### 1.1 目标用户画像

| 维度 | 描述 |
|------|------|
| **身份** | 25-40岁职场人群，兼顾工作与生活多重角色 |
| **痛点** | 时间碎片化、记录需求旺盛但无系统性工具、人际关系维护被动、生活节律不稳定 |
| **使用场景** | 每日通勤/午休碎片时间快速记录、周末规划复盘、碎片化物品位置查询 |
| **心理诉求** | 零焦虑、温和引导、不制造压力、让记录成为习惯而非负担 |

### 1.2 核心模块架构

系统由 **6大功能域** 组成，通过底部导航和全局搜索串联：

#### A. 日常记录域（记录引擎）

| 模块 | 页面 | 核心能力 |
|------|------|----------|
| 任务管理 | home / nlp | 时间轴拖拽、NLP智能解析、固定/弹性双模式、循环任务克隆 |
| 目标体系 | goal | 年/季/月/周OKR层级、反目标(Anti-Goals)、If-Then应对预案 |
| 打卡系统 | checkin | ABC三档(完美日/基础日/兜底日)、月度热力图、高光时刻 |
| 习惯追踪 | habits | 每日勾选、Streak连续天数、12图标、周完成点阵 |

#### B. AI辅助域（智能引擎）

| 模块 | 能力 |
|------|------|
| NLP智能录入 | 自然语言转为结构化任务（时间/标签/优先级/循环/时长） |
| AI深度洞察 | 基于7天数据的趋势分析、精力评估、建议生成 |
| AI复盘 | 自动生成周回顾摘要 |
| 话术风格 | 陪伴型/效率型/简洁型三档切换 |

#### C. 作息管理域（节律引擎）

| 模块 | 能力 |
|------|------|
| 番茄钟 | 标准(25min)/冲刺(10min)/深度(50min)三模式 |
| 精力追踪 | 每日精力状态可视化、本周趋势柱状图 |
| 时间分布 | 工作/生活/成长/休息四象限分析 |
| 任务提醒 | 30分钟到期通知、过期警告 |

#### D. 人脉关系域（社交引擎）

| 模块 | 能力 |
|------|------|
| 人脉管理 | 联系人CRUD、关系/公司/备注字段 |
| 互动提醒 | （规划中）互动频率追踪、生日/周期提醒 |

#### E. 知识辅助域

| 模块 | 能力 |
|------|------|
| 智能记忆 | 物品位置管理、分类筛选、全文搜索 |
| 全局搜索 | 跨4模块实时搜索、关键词高亮、搜索历史 |
| 数据统计 | 趋势图、番茄钟统计、目标进度条、月度热力图 |

#### F. 系统支撑域

| 模块 | 能力 |
|------|------|
| 用户认证 | 注册/登录/锁屏/SHA-256加密/密码重置 |
| 数据安全 | JSON/CSV/Excel导入导出、版本迁移、备份恢复、数据清理 |
| 通知系统 | Browser Notification、每日打卡提醒、任务截止提醒 |
| PWA | Service Worker离线缓存、安装提示 |

### 1.4 设计哲学

系统的设计围绕三个核心原则：

1. **零焦虑** — ABC三档打卡确保"不完美也是成功"。兜底日(C档)让用户在最忙的日子里仍然保持连续性。
2. **温和引导** — AI 建议采用"陪伴型"默认话术，避免命令式语气。所有提醒均可关闭。
3. **数据主权** — 100%本地存储，用户完全掌控自己的数据。导入导出开放标准格式。

### 2.2 差异化优势

**1. 物品位置记忆 — 独有功能，填补市场空白**

Notion / Flomo / Obsidian / 滴答清单均无此功能。解决"护照放哪了""备用钥匙在哪"的高频刚需。6分类 + 全文搜索 + 位置标注 + 存放时间记录，填补了个人生活管理工具中的一个明显空白。

**2. 零焦虑设计哲学 — 体验差异化**

ABC三档打卡（完美日 / 基础日 / 兜底日）是行业首创。Notion无打卡功能，滴答清单只有完成/未完成二元状态。兜底日（C档，仅需15分钟）让用户在极端忙碌时仍能保持"不中断"的心理安全感——这是对职场人群心理需求的精准洞察。

**3. 全本地存储 + PWA — 隐私制高点**

100% localStorage 存储，无服务器、无云端同步、无第三方数据访问。Obsidian同样本地化但为桌面端优先，移动端体验弱。本系统PWA可安装到手机主屏幕，离线可用。

**4. NLP + 本地解析双引擎 — 效率极致**

一句话"明天下午开会 #工作 重要 2小时"即可自动解析为结构化任务。Flomo支持标签但无结构化解析能力。AI降级策略：API不可用时自动切换本地正则解析，保证功能不中断。

**5. 全功能一体化 — 去碎片化**

用户无需在 Notion（笔记）+ 滴答清单（任务）+ Day One（日记）+ 习惯App 之间切换。一个PWA覆盖生活OS全部场景，减少工具切换的认知负担。

### 2.3 现存不足

| 不足 | 严重度 | 竞品对比 |
|------|--------|----------|
| 无多端同步 | 高 | Notion / Obsidian 有云同步 |
| 无日记/随笔模块 | 中 | Day One 核心就是日记 |
| 无协作分享 | 中 | Notion 强项 |
| 无日历视图 | 中 | 滴答清单有 |
| 无API/插件生态 | 低 | Obsidian 社区生态强大 |
| localStorage 5MB上限 | 中 | IndexedDB 可解决 |
| 仅移动端优先UI | 低 | Notion 全平台适配 |

## 三、核心价值与问题解决能力

### 3.1 解决的核心痛点

| 用户痛点 | 解决方案 | 量化效果 |
|----------|----------|----------|
| 碎片时间无法高效记录 | NLP智能解析 + 首页FAB快捷面板 | 输入到解析到保存 < 3秒 |
| 缺乏生活秩序感 | OKR目标体系 + ABC打卡 + 习惯Streak | 从年目标拆解到每日微习惯 |
| 人际关系被动维护 | 人脉管理 + 互动提醒（规划中） | 主动触达而非被动等待 |
| "东西放哪了" | 智能记忆物品位置管理 | 秒级检索，终结翻箱倒柜 |
| 番茄钟无法坚持 | 冲刺模式(10min) + 深度模式(50min) | 降低启动门槛，覆盖不同精力状态 |
| 数据丢失焦虑 | 3格式导出 + 版本迁移 + 5备份 | 数据完全可控 |
| 工具切换疲劳 | 一站式生活OS | 1个PWA替代4-5个独立App |

### 3.2 价值主张

> 「个人之旅」不是另一个待办清单，而是一个**温和的生活副驾驶**——
> 不催促、不制造焦虑、不要求完美，
> 只在碎片时间里帮你抓住重要的东西：任务、目标、习惯、人脉、物品。

### 3.3 核心差异化总结

与其他工具相比，「个人之旅」在三个维度上建立了独特优势：

- **功能广度**：唯一同时覆盖任务、目标、习惯、人脉、物品记忆的生活OS
- **体验深度**：ABC打卡、NLP智能输入、AI洞察——这些功能在同类产品中要么缺失要么粗糙
- **隐私高度**：全本地 + PWA + 开放格式导出 = 用户对数据拥有绝对主权

## 四、下一步迭代方向建议

### 4.1 产品路线图总览

```
Phase 1 (V1.9-V2.0)     Phase 2 (V2.1-V2.2)     Phase 3 (V2.3-V2.4)     Phase 4 (V2.5+)

├── 情绪趋势             ├── 生活节律分析         ├── 数据压缩            ├── 财务追踪
```

#### Phase 1: 体验深化（V1.9-V2.0）

| 优先级 | 功能 | 详细描述 | 预期效果 |
|--------|------|----------|----------|
| P0 | **日记/随笔模块** | 富文本编辑 + 情绪标签(😊😐😔) + 关联当日任务和打卡 | 补齐最大功能缺口，对标 Day One 核心体验 |
| P0 | **情绪趋势追踪** | 基于每日情绪标签生成月度情绪曲线和洞察 | 让用户看到自己的情绪周期，AI可给出调节建议 |
| P1 | **日历视图** | 任务/打卡/习惯在月视图上可视化聚合 | 对标滴答清单，提升时间感知 |
| P1 | **深色模式对比度修复** | 确保 WCAG AA 标准（4.5:1 对比度） | 提升夜间使用舒适度和可访问性 |
| P2 | **首页时间轴折叠** | 超过10个任务时自动折叠，点击展开 | 减少长列表滚动疲劳 |

#### Phase 2: AI 深度化（V2.1-V2.2）

| 优先级 | 功能 | 详细描述 | 预期效果 |
|--------|------|----------|----------|
| P0 | **个性化建议引擎** | 基于30天历史数据学习用户行为模式（高效时段、拖延模式、精力周期） | 从通用建议进化为"懂你"的个性化建议 |
| P1 | **生活节律分析** | 分析番茄钟完成时间 + 打卡档位 + 任务完成率的交叉关联 | 推荐最佳工作时段和休息策略 |
| P1 | **智能习惯推荐** | 根据打卡档位分布和任务完成率，推荐应建立的新习惯 | 从"用户自己想习惯"到"AI帮你发现该养成的习惯" |
| P2 | **周报自动生成** | 每周自动生成摘要（完成率、番茄钟统计、习惯追踪、情绪趋势） | 可导出为图片分享，增加社交传播属性 |

#### Phase 3: 数据升级（V2.3-V2.4）

| 优先级 | 功能 | 详细描述 | 预期效果 |
|--------|------|----------|----------|
| P0 | **IndexedDB 迁移** | 从 localStorage 迁移到 IndexedDB，突破5MB上限 | 支持多年数据积累，消除存储焦虑 |
| P1 | **数据压缩** | 导入导出时支持 gzip 压缩 | 备份文件体积减小70%+ |
| P1 | **人脉互动追踪** | 记录联系频率、上次互动时间、生日提醒 | 从"存了联系方式"到"主动维护关系" |
| P2 | **WebDAV/云盘同步** | 支持坚果云等 WebDAV 服务的自动备份 | 在不引入服务器的前提下实现"伪同步" |

#### Phase 4: 生态扩展（V2.5+）
